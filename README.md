# Router Health Assistant (PoC)

This is a lightweight, engineer-side automation tool intended for CX/TAC workflows.

## Features
- On-demand per-device routing health checks
- Rule-based issue detection
- Human-readable explanations
- Modular device adapter design

## Scope
- Not a monitoring platform
- Not autonomous
- Focused on individual devices and cases

## Tech Stack
- Python
- Netmiko (planned)
- CLI / JSON output

## Status
Work in progress (PoC)
